# I3AM_AHR
This is the multi-omics data of the I3AM_AHR project.

Fig1 dataset: Bulk RNA-seq TPM.gene.annotated.txt,Bulk RNA-seq count.csv
MCE.rds and RT.rds: Raw sparse matrix for snRNA-seq in Fig2
Human_metabolics_data.levels.xlsx and Mouse_RT_MCE_metabolics_data.xlsx: Metabolomic Results of Human and Mouse Blood
